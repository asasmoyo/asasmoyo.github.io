Setup PostgreSQL on Ubuntu 16.04

Install PostgreSQL By default, Ubuntu 16.04 instance has already postgresql package, but it is pretty out of date. You can still install latest version available (currently 9.6) by adding PostgreSQL repo into your ubuntu instance. First create a file at /etc/apt/sources.list.d/postgres.list with this content deb http://apt.postgresql.org/pub/repos/apt/ xenial-pgdg main Then run these commands to install PostgreSQL signing key and also install PostgreSQL 9.6 wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add - sudo apt-get update sudo apt-get install postgresql-9.
